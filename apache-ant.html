Apache ANT

This page contains QnA on Apache Ant from TutorialsPoint.com

Ant is a java based build tool developed by Apache for the purpose of automating the build and deployment process.

Another Neat Tool

Following are the activities that are done by a developer manually after writing the application code:-
  1. Compiling the application code
  2. Packaging the binaries (class files)
  3. Deploying the packaged binaries to test server. (Note that this is not the PROD server)
  4. Testing the application code on the test server
  5. Copying the application code from test server to the prod server.
All of the above activities were done by the developer manually onne after the other. With automated build and deployment tool like Apache Ant, there is no further need to do all these steps manually.
Follow the steps below:-

1. Goto website https://ant.apache.org/

2. Click on the link to download Ant in binary form "https://ant.apache.org/bindownload.cgi."



3. Click on tar.gz link (since you are using macbook).



4. Tarred (zipped) file gets downloaded onto your macbook.



5. Double click on the tarred file to extract it. You get a ant folder.



6. Open terminal and run the command:- export ANT_HOME=/Users/ngksh/Downloads/apache-ant-1.9.14 where "/Users/ngksh/Downloads/apache-ant-1.9.14" denotes the path to the extracted folder in step 5.

7. Run command:- export PATH=${PATH}:${ANT_HOME}/bin. Note that "bin" folder is present within the extracted folder.

8. Run command:- ant -version. You should see the version of ant installed on your macbook.
